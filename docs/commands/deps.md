# `pnpm playbook deps`

Print module dependency relationships from `.playbook/repo-index.json`.

## Usage

- `pnpm playbook deps`
- `pnpm playbook deps workouts`
- `pnpm playbook deps --json`

## Behavior

- Uses the repository index generated by `pnpm playbook index`.
- `pnpm playbook deps` prints all indexed modules and their direct dependencies.
- `pnpm playbook deps <module>` prints direct dependencies for one module.
- Returns a deterministic error when the requested module does not exist.

## JSON contract

```json
{
  "schemaVersion": "1.0",
  "command": "query",
  "type": "dependencies",
  "module": "workouts",
  "dependencies": ["auth", "db"]
}
```
